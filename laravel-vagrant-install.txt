Software: 
  - VirtualBox
  - Git
  - Vagrant
  - Composer
  - Laravel homestead

Pasos: 
  - Instalar VirtualBox
  - Instalar Git
  - Instalar Vagrant
  - En documentacion laravel instalación laravel/homestead:
    - vagrant box add laravel/homestead
    - git clone https://github.com/laravel/homestead.git Homestead
    - bash init.sh (creara el archivo homestead.yaml (desde homestead))
    - ssh-keygen -t rsa -C "you@homestead"
    - En Homestead.yaml configuracion rutas carpetas
    - Crear carpeta Code en carpeta personal
    - Cambiar homestead.app por homestead.test
    - En fichero /etc/hosts/hosts añadir ip de Homestead.yaml a homestead.test 

  ESTO SERIA PARA CREAR UN NUEVO PROJECTO
  - Instalar composer en caso de no tenerlo ya instalado
  - composer create-project laravel/laravel --prefer-dist (desde /Code)
  - Se tendria que haber creado el projecto laravel  

  TENGO QUE CLONAR EL RPOJECTO DE GITHUB EN MI CARPETA CODE

  UNA VEZ TODO ESTE LISTO
  - vagrant up (desde carpeta homestead)
  - vagrant ssh
